Problem #4

Input: n1, n2, and n3

Output: maxNum

Example Input: 5, 7, 3

Expected Output: 7

Example Input: 1, 8, 10

Expected Output: 10

View solutions only after trying to solve the problem on your own.

  1. Let maxNum
  2. Input n1, n2, and n3
  3. maxNum = n1
  4. If maxNum > n2 and maxNum > n3, go to step 8
  5. maxNum = n2
  6. If maxNum > n3, go to step 8
  7. maxNum = n3
  8. Output maxNum

Problem #6

A user wants to buy an item and pay thru digital wallet. Write a program that outputs the remaining balance after buying the item. But if the user’s digital wallet is not enough to pay for the item, output “Insufficient Funds”.

Input: itemPrice, walletBal

Output: walletBal or "Insufficient Funds"

Example Input: itemPrice = 300, walletBal = 500

Expected Output: 200

Example Input: itemPrice = 100, walletBal = 20

Expected Output: "Insufficient Funds"

View solutions only after trying to solve the problem on your own.

  1. Input itemPrice, walletBal
  2. If walletBal >= itemPrice, go to step 4
  3. Output "Insufficient Funds", end
  4. walletBal = walletBal - itemPrice
  5. Output walletBal

Problem #10

Write a program that takes an array of numbers and outputs their sum

Input: nums[] and size

Output: sum of nums

Example Input: [5, 8, 10]

Expected Output: 23

Example Input: [2, 1, 5, 3]

Expected Output: 11

View solutions only after trying to solve the problem on your own.

  1. Let sum, index
  2. Input nums[], size
  3. sum = 0, index = 0
  4. sum = sum + nums[index]
  5. index == size - 1, go to step 7
  6. index = index + 1, go back to step 4
  7. Output sum
